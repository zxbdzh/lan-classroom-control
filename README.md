# 局域网机房控制系统

面向学校机房/培训机构的局域网管控系统，支持教师端控制学生端黑屏、屏幕广播、网络管控、文件分发等功能。

## 功能特性

### 核心功能
- **屏幕广播**：教师屏幕实时广播到学生端，广播期间禁用学生键鼠
- **黑屏肃静**：一键黑屏 + 键鼠锁定，维持课堂纪律
- **网络管控**：禁用/启用学生上网（系统防火墙级别）
- **文件分发**：教师一键下发文件给全体或指定学生
- **自动发现**：UDP 广播自动发现教师端，学生端一键接入

### 技术特点
- 跨平台架构（核心在 Windows 上完整实现，Linux/macOS 可运行基础功能）
- 自定义 TCP 协议，支持粘包/半包处理
- AES-256 加密通信
- 屏幕帧 JPEG 压缩 + 增量传输优化
- 多线程架构，支持百台级学生并发

## 安装

### 环境要求
- Python 3.10+
- Windows 10/11（完整功能）
- 管理员权限（网络管控功能需要）

### 安装依赖
```bash
pip install -r requirements.txt
```

## 快速开始

### 启动教师端
```bash
python teacher/main.py
```

教师端启动后会自动：
- 监听 TCP 端口 9528
- UDP 广播（端口 9527）宣告存在
- 显示主界面，左侧学生列表，右侧屏幕墙

### 启动学生端
```bash
python student/main.py
```

学生端启动后会自动：
- 监听 UDP 广播寻找教师端
- 发现教师端后自动连接
- 在系统托盘显示图标

> **注意**：学生端需要管理员权限运行，否则网络管控和键鼠禁用功能可能受限。

## 使用说明

### 教师端界面
- **工具栏按钮**：
  - `开始广播` / `停止广播`：屏幕广播控制
  - `黑屏肃静` / `解除黑屏`：黑屏控制
  - `禁用上网` / `解除禁网`：网络控制
  - `发送文件`：文件分发

- **学生列表**（左侧）：
  - 按分组显示所有学生
  - 支持多选操作
  - 右键菜单：重命名、分组、单独控制

- **屏幕墙**（右侧）：
  - 网格展示所有学生缩略图
  - 显示在线状态和当前控制状态

### 学生端
- 系统托盘图标显示运行状态
- 黑屏/广播时全屏覆盖，键鼠禁用
- 文件接收完成后弹窗通知

## 端口说明

| 端口 | 协议 | 用途 |
|------|------|------|
| 9527 | UDP | 教师端发现广播 |
| 9528 | TCP | 主控制通道（指令+心跳+文件） |

可通过配置文件修改端口。

## 配置文件

配置文件位置：`~/.lan_classroom/config.json`

主要配置项：
```json
{
  "teacher": {
    "udp_broadcast_port": 9527,
    "tcp_port": 9528,
    "heartbeat_interval": 5,
    "heartbeat_timeout": 15,
    "broadcast_fps": 20,
    "broadcast_quality": 70
  },
  "student": {
    "auto_connect": true
  }
}
```

## 安全说明

1. **仅限局域网使用**，不要暴露到公网
2. 学生端需管理员权限安装运行
3. 所有操作在教师端有日志记录
4. 通信通道预留加密接口（可扩展启用）
